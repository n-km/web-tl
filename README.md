# web-tl
an responsive web-template
## File-Structure
```FILE-STRUCTURE.TXT
web-tl/
├─ assets/
    ├─ icons/
├─ content/
├─ parts/
    ├─ footer.php
    ├─ header.php
├─ scripts/
    ├─ addons/
    ├─ web-tl/
        ├─ main.js
├─ styles
    ├─ addons/
    ├─ web-tl/
        ├─ main.css
        ├─ menu.css
└─ index.php
```
